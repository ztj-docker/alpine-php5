# docker-ztj1993-alpine-php5

alpine php5 基础映像，安装了所有可用扩展。

## 相关链接
- [GitHub](https://github.com/ztj-docker/alpine-php5)
- [DockerHub](https://hub.docker.com/r/ztj1993/alpine-php5)

## 使用教程
```
docker run -ti --rm ztj1993/alpine-php5
docker run -ti --rm ztj1993/alpine-php5 php -v
```
